# PromptTranslater

A custom extension for [AUTOMATIC1111/stable-diffusion-webui](https://github.com/AUTOMATIC1111/stable-diffusion-webui) which helps to write prompt in your native language, translating it into English for you.

## Using

In order to use this extension, select the text that you need to translate and press Alt+t or the Å¦ button, to the right of the tools. You can also not select the text, but just click and then the entire prompt will be translated, without affecting the negative prompt.
